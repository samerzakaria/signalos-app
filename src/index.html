First project journey

1. Choose a project folder
Pick the folder you will actually work in. SignalOS stores its local state inside that project.
2. Connect AI
Select a provider, fetch models, choose a model, and test the connection. Keys stay in the operating system credential manager.
3. Set up project
Run setup from Chat. Confirm the Project result panel shows the expected local files.
4. Work from Dashboard
Use the current gate, next action, project files, and command results to move one step at a time.

Project templates

Recovery

AI not connected
Open Settings, fetch models, select a model, then save and test the connection.
Engine failed
Open Settings, restart the engine, then run status again. Copy diagnostics if it still fails.
Setup unclear
Run status and inspect Project result. Missing files are shown directly in the artifact list.
