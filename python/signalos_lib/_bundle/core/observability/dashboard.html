SignalOS Core — metrics sidecar dashboard

This page is a placeholder. The real dashboard is a single static HTML document generated from per-session metrics.jsonl files under .signalos/sessions/. No JavaScript, no CDN, no build step.

Regenerate it with:

python3 core/observability/render-dashboard.py \
  --output core/observability/dashboard.html

The renderer is Python stdlib only. Given identical input bytes, two invocations produce byte-identical HTML — this makes the snapshot safely git-trackable. Concept adapted from a5c-ai/babysitter (MIT); no source code copied.

See core/execution/skills/observability-dashboard/SKILL.md for the read-side redaction contract, chart-extension rules, and jq recipes for reading raw metrics.
