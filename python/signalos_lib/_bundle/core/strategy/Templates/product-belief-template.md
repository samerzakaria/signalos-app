## Entry criteria

A Product Belief exists only after all daemon-mode entry criteria (Constitution §12.2) are satisfied:

- [ ] At least 3 consecutive Waves returned "Keep" on the same Wave-level Belief with no material amendments
- [ ] T3 surface list has been stable for 3 Waves
- [ ] Backlog is queue-like (refined items with acceptance lines)
- [ ] PO + PE commit to quarterly product retro
- [ ] Client signed (if engagement model requires)

*If any box is unchecked, the product stays in `fresh-wave` mode and this file is not authored.*
